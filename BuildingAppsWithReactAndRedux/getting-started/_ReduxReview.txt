
What is the React-Redux library?
- It provides a number of helper methods that make it
easier to connect Redux to your React application.

Redux is a JS Framework that is an alternative to React.
- False

Redux expects you to implement features a specific way, otherwise your application
will not work as expected.
- True. Redux is optionated. Following a consistent pattern is easier for developers
familiar with Redux to understand the application structure of any Redux app.

Do React applications require that you use Redux?
- No, make sure your app will benefit from using Redux.

React maintains and manages application *state* in a scaleable and predictable way.

Why does Redux make your application more reliable and stable?
- Redux provides a single source of truth for your application, making it
  easier to keep application state in sync and distribute data.
